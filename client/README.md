# Клиентская часть

Фронтенд для панели модерации Авито. React 18 + TypeScript + Vite.

## Команды

- `npm run dev` — локальная разработка (порт 5173).
- `npm run build` — сборка.
- `npm run lint` — ESLint + Prettier.

API адрес задаётся переменной `VITE_API_URL` (по умолчанию `http://localhost:3001/api/v1`).
