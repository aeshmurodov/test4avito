# Авито — панель модерации

Веб‑приложение для модерации объявлений, соответствующее тестовому заданию. Клиент на React 19 + TypeScript + Vite, сервер — готовый мок API из папки `server`.

## Запуск

1. Установите зависимости:
   ```bash
   cd server && npm install
   cd ../client && npm install
   ```
2. Запустите API (порт 3001):
   ```bash
   cd server && npm start
   ```
3. Запустите клиент:
   ```bash
   cd client && npm run dev
   ```
   По умолчанию клиент ожидает API на `http://localhost:3001/api/v1`. Можно переопределить переменной `VITE_API_URL`.

## Выбор технологий
- **React Router 7** — маршрутизация `/list`, `/item/:id`, `/stats`.
- **@tanstack/react-query** — запросы к API с отменой по сигналу и автоматическим кеш‑инвалидированием после модерации.
- **MUI** — готовые UI-компоненты, быстрый отклик на требования макета и тёмная тема.
- **Zustand** — лёгкий стейт для темы и выбора объявлений в bulk-операциях.
- **TypeScript** — типобезопасность API/сущностей объявлений и статистики.
- **Recharts** — графики активности, распределения решений и категорий на странице статистики.

## Функциональное покрытие
- Список объявлений с фильтрами (статус, категория, цена, поиск), сортировками и пагинацией 10/страница с указанием общего количества.
- Карточки объявлений с изображением, категорией, ценой, датой, статусом и индикатором срочности; чекбоксы для bulk-операций.
- Детальная страница: галерея 3+, характеристики, продавец, история модерации; действия «Одобрить», «Отклонить», «Вернуть на доработку» с шаблонами причин.
- Навигация «Назад к списку», «Пред/След» и горячие клавиши: `A` — одобрить, `D` — отклонить, `←/→` — переключение карточек, `/` — фокус поиска на списке.
- Статистика с выбором периода (сегодня/7/30 дней), картами метрик, графиком активности, распределением решений и категориями.

## Тесты и качество
- ESLint + Prettier для стиля кода (скрипт `npm run lint` в `client`).
- Реакт-квери использует `signal` для отмены запросов при уходе со страницы.
